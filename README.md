# To-Do List Application

Overview

A responsive and interactive To-Do List application built using HTML, CSS, and JavaScript. The project focuses on JavaScript logic, DOM manipulation, event handling, state management, and local storage to create a complete task management experience.

Features

 CRUD Operations:

* Create new tasks
* View existing tasks
* Edit task details
* Delete tasks

 Task Management:

* Mark tasks as completed or active
* Track total, completed, and remaining tasks
* Filter tasks by:

  * All
  * Active
  * Completed

 Data Persistence:

* Tasks are stored using browser localStorage
* Data remains available after page refresh or browser restart

 User Interface:

* Responsive design
* Dynamic task rendering
* Empty-state message when no tasks are available
* Active filter highlighting

 Technologies Used:

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API
